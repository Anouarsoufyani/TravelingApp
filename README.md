# Traveling

Application Android de voyage.

## Lancer le projet

1. Ouvrir le projet avec Android Studio.
2. Attendre la synchronisation Gradle.
3. Lancer l’application sur un appareil Android virtuel.
4. Un appareil physique est préférable si possible.
5. Exécuter l’application avec le bouton run

Le projet utilise déjà `app/google-services.json` pour Firebase.
Nous avons testé l’application sur un Redmi 15 et tout fonctionnait correctement.
